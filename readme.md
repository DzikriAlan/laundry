# Coding Test Penggajian - Dzikri Alan

## Versi
Node.js : v16.20.1, npm : 8.19.4, composer: 2.6.5

## Installation
```bash
cp .env.example .env
```

Rubah bagian DB di file .env menjadi seperti dibawah ini
```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=coding_test_penggajian
DB_USERNAME=root
DB_PASSWORD=
```

## Usage

```python
# install composer
composer install

# lalu
php artisan key:generate

# install npm
npm install

# dan jalankan
npm run watch
php artisan serve

# login admin
username: dzikri@gmail.com
password: password

# login pegawai
username: auliaharvy@gmail.com
```
